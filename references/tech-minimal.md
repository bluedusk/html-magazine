# Tech — Wired / MIT Technology Review

## Editorial Voice

Sharp, precise, slightly irreverent. Confident without being arrogant. Opens with a fact or bold claim, never a scenic description. Numbers are first-class citizens — stats deserve visual prominence. Headlines make declarative statements. Pull quotes are provocative predictions. Closes by looking forward.

Kickers: DEEP DIVE, THE BIG IDEA, INFRASTRUCTURE, DATA

## Visual Tone

A cutting-edge tech publication. Sharp, precise, high-contrast. Everything feels like a well-designed developer tool. The surface behind the magazine is near-black. Pages are clean and bright against the dark surface. No soft edges, no rounded corners, no warmth — this is geometry and whitespace.

Typography is modern and commanding. Bold sans-serif headlines with extremely tight tracking. Clean sans-serif body text. Monospace font for data, stats, and technical elements. No drop cap — the first line is oversized and bold instead.

Two accent colors — one hot/electric for primary accents, one neon/bright for data highlights. A gradient bar or rule using both colors is a signature element. Data callout boxes have dark backgrounds with large bright numbers. Everything is high contrast.

Images are technical and abstract. Close-ups, circuits, code, data visualization. The magazine feels sharp, futuristic, and confident.

## Media Mood

Technical, abstract, high-contrast. Futuristic, digital, precise. Cover uses semi-transparent image behind dark overlay. Body images get thin accent borders. Avoid nature, warm tones, soft aesthetics.
